
Hardware Requirements
The following hardware requirements are necessary for efficient execution of the
proposed system:
• Processor: Intel Core i5 or higher
• RAM: Minimum 8 GB (16 GB recommended)
• Storage: Minimum 20 GB free disk space
• GPU (Optional): NVIDIA CUDA-enabled GPU for faster deep learning training
• Network : Stable broadband connection
3.6.2 Software Requirements
The software tools and technologies required for this project include:
• Operating System: Windows / Linux / macOS
• Programming Language: Python 3.8 or above
 Blockchain Platform: Ganache
• Development Tools: Jupyter Notebook, VS Code,Googel Collab
• Frameworks: TensorFlow, Keras, Flask
• Libraries: Scikit-Learn, NumPy, Pandas, Matplotlib, seaborn, PyTorch
